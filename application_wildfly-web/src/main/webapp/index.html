Bienvenido a Finance App

Esta es una aplicación diseñada como parte del Trabajo de Fin de Máster (TFM) de Bryan Beltrán y Gustavo Contreras. El objetivo es implementar un prototipo de Infraestructura como Código (IaC) que facilite la gestión y despliegue de servicios financieros.

En esta aplicación se demostrará la viabilidad de automatizar tareas críticas relacionadas con servicios financieros, utilizando tecnologías modernas como WildFly, Java EE, y estrategias de IaC.

Características principales:

  • Gestión de servicios financieros.
  • Automatización mediante IaC.
  • Diseño modular y escalable.
